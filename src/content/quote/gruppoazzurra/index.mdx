2. **ICCA** - Sito vetrina/portfolio edilizia
3. **WD Supermercati** - Sito retail complesso con CMS e automazioni

## Infrastruttura

- [ ] Creazione VPS
- [ ] Aggiornamento
- [ ] Hardening base (UFW + disable root login)
- [ ] Configurazione reverse proxy e HTTPS (Caddy)
- [ ] Impostazione dominio (redirect alla VPS)
- [ ] Configurazione ambienti dockerizzati

### Sono incluse:

- [ ] 1 settimana di assistenza sistemistica per eventuali problemi di configurazione collegati ai servizi forniti di base a partire dalla consegna del progetto
- [ ] Documentazione della configurazione e procedure di base

- Configurazione di sistema di backup per i 3 ambienti dockerizzati (```+250€```)
