# WD Supermercati - Documento di Sintesi

## Panoramica
Sito retail complesso orientato alle offerte settimanali. Focus su punti vendita, sistema offerte aggiornabile settimanalmente, e automazione invio newsletter/WhatsApp.

**Tecnologia**: Next.js + CMS  
**Tipo**: Retail/CMS

---

## Pagine da Sviluppare

### Homepage
- Header con menu (Punti vendita, Offerte, News, Contatti) e CTA "Iscriviti alle offerte"
- Hero con slider/immagine e CTA "Vedi tutte le offerte"
- Sezione "Offerte della settimana" (anteprima top 6-8 offerte)
- Sezione punti vendita (anteprima con link "Vedi tutti")
- Sezione News/Comunicazioni (ultime 3-4, opzionale)
- Footer con contatti e link utili

### Sezione "Iscriviti alle Offerte"
- Form iscrizione newsletter (Email, Consenso privacy, Checkbox marketing opzionale)
- Double opt-in (conferma email)
- Pulsante "Segui il canale", QR code, link diretto, istruzioni
- Spiegazione vantaggi iscrizione
- Messaggio conferma iscrizione

### Sezione "Contatti"
- Informazioni contatto (email, telefono, indirizzo sede)

### Pagina "Offerte"
- Filtri categoria (Tutte, Freschi, Dispensa, Bevande, Casa, Igiene, ecc.)
- Grid offerte responsive con card (immagine, titolo, categoria, prezzi, badge "TOP", note validità)
- Badge "Offerta TOP" per evidenziare offerte principali
- Indicatori validità (data inizio/fine)
- Paginazione o infinite scroll
- Note "fino a esaurimento scorte" (se presente)

### Pagina "Dettaglio Offerta / Volantino"
- Scaricamento volantino PDF o grid completa offerte settimana
- Condivisione social

### Pagina "Punti Vendita"
- Mappa Google Maps con marker cliccabili per ogni punto vendita
- Lista punti vendita con card (nome, indirizzo, orari, telefono, servizi, foto, link dettaglio)
- Filtro ricerca per località con autocomplete
- Vista mappa / vista lista (toggle)

### Pagina "Dettaglio Punto Vendita"
- Informazioni complete punto vendita
- Google Maps embed con marker
- Orari dettagliati
- Servizi disponibili (macelleria, panetteria, ecc.)
- Gallery foto (opzionale)
- CTA "Come arrivare" (link Google Maps)

### Pagina "News/Comunicazioni"
- Lista comunicazioni/news
- Filtri per categoria (se necessario)

### Pagine Legali
- Privacy Policy
- Cookie Policy
- Termini e Condizioni (se necessario)

## Note per lo Sviluppo

### CMS e Backend
- **Directus CMS**: Configurazione per gestione offerte, punti vendita, news
- **Upload multiplo**: Sistema veloce per caricamento offerte settimanali
- **Gestione immagini**: Ottimizzazione automatica (webp, resize)

### Sistema Offerte
- **CRUD offerte**: Creazione, modifica, eliminazione offerte
- **Bulk upload**: Possibilità upload multiplo offerte
- **Validità date**: Gestione date inizio/fine validità
- **Badge "TOP"**: Sistema per evidenziare offerte principali
- **Filtri categoria**: Filtri dinamici per categoria
- **Ricerca**: Ricerca prodotti

### SEO e Performance
- **SEO**: Title/meta per ogni pagina, H1-H2 corretti, URL pulite, sitemap.xml
- **Performance**: Immagini ottimizzate (webp), lazy load, caching, CDN (se necessario)
- **Responsive**: Mobile-first, test su dispositivi vari

### Sicurezza
- **HTTPS**: Certificato SSL
- **Validazione**: Validazione input form, sanitizzazione dati
