# Preventivo - WD Supermercati

**Tipo Progetto**: Sito retail complesso con CMS e automazioni  
**Tecnologia**: Next.js + Directus CMS  
**Pagine Principali**: 7-8 pagine (Homepage, Offerte, Dettaglio Offerta, Punti Vendita, Dettaglio Punto Vendita, Iscriviti alle Offerte, News, Pagine Legali)

---

## Tariffa Applicata

**Ruolo**: UX Engineer (Fullstack + Designer)  
**Tariffa Oraria**: €50/ora  
**Tariffa Giornaliera**: €350/giorno (7 ore/giorno)

---

## Dettaglio Costi per Sezione

### 1. Infrastruttura e Setup Ambiente

**Attività**:
- Setup VPS Hetzner (configurazione base Ubuntu LTS)
- Configurazione Docker e Docker Compose
- Setup reverse proxy Caddy con SSL automatico
- Configurazione firewall e sicurezza base (UFW, Fail2Ban)
- Configurazione DNS Cloudflare
- Setup ambiente di sviluppo locale
- Setup database PostgreSQL per Directus
- Setup storage MinIO per file multimediali

**Tempo Stimato**: 12 ore  
**Costo**: €600

---

### 2. Design - Raccolta e Sistemazione Asset

**Attività**:
- Analisi e verifica asset ricevuti (loghi, immagini punti vendita, testi)
- Ottimizzazione immagini punti vendita per web (formato webp, resize multipli)
- Organizzazione immagini per punto vendita (gallerie)
- Preparazione loghi in varianti (light/dark)
- Creazione favicon set completo
- Preparazione font per uso web
- Organizzazione asset per sviluppo
- Preparazione immagini placeholder per offerte

**Tempo Stimato**: 12 ore  
**Costo**: €600

### 6. Implementazione Frontend

## Note

- I costi non includono IVA (se applicabile)
- I tempi sono stimati considerando che il cliente fornisca tutti gli asset e contenuti richiesti nei tempi previsti
- Eventuali modifiche sostanziali al design o alle funzionalità dopo l'approvazione dei wireframe potrebbero comportare costi aggiuntivi
- **Provider Email Marketing**: Il preventivo include integrazione con provider standard (Mailchimp/Sendinblue/SendGrid). Costi mensili del provider non inclusi nel preventivo.
- **WhatsApp**: Implementazione iniziale con canale semplice (QR code). Integrazione API WhatsApp Business richiederebbe servizi esterni autorizzati da Meta con costi aggiuntivi significativi.
- **CMS Directus**: Il cliente potrà gestire autonomamente offerte, punti vendita e news tramite interfaccia admin dopo la consegna.
- Costi infrastruttura hosting (VPS Hetzner) non inclusi nel preventivo (vedi documento infrastruttura.md)
- Il numero iniziale di offerte e punti vendita influisce sui tempi di setup CMS (stima basata su 20-30 offerte iniziali e 5-10 punti vendita)
- **Automazione Newsletter**: Richiede configurazione iniziale e test. Il sistema invierà automaticamente le offerte settimanali agli iscritti.
