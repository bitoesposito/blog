# Preventivo - Gruppo Azzurra

**Tipo Progetto**: Sito istituzionale corporate  
**Tecnologia**: Astro (sito statico)  
**Pagine Principali**: 5 pagine (Homepage, Il Gruppo, Le Aziende, Contatti, Pagine Legali)

---

## Dettaglio Costi per Sezione

### 1. Infrastruttura e Setup Ambiente

**Attività**:
- Setup VPS Hetzner (configurazione base Ubuntu LTS)
- Configurazione Docker e Docker Compose
- Setup reverse proxy Caddy con SSL automatico
- Configurazione firewall e sicurezza base (UFW, Fail2Ban)
- Configurazione DNS Cloudflare
- Setup ambiente di sviluppo locale

**Tempo Stimato**: 8 ore  
**Costo**: €400

---

### 2. Design - Raccolta e Sistemazione Asset

**Attività**:
- Analisi e verifica asset ricevuti (loghi, immagini, testi)
- Ottimizzazione immagini per web (formato webp, resize)
- Preparazione loghi in varianti (light/dark)
- Creazione favicon set completo
- Preparazione font per uso web
- Organizzazione asset per sviluppo

**Tempo Stimato**: 6 ore  
**Costo**: €300

---

### 3. Wireframing

**Attività**:
- Creazione wireframe low-fidelity per tutte le pagine
- Definizione struttura layout responsive
- Annotazioni flussi utente principali
- Wireframe mobile, tablet, desktop
- Revisione e iterazione wireframe

**Pagine da wireframare**: 5 pagine principali  
**Tempo Stimato**: 10 ore  
**Costo**: €500

---

### 4. Design System e Prototipazione

**Attività**:
- Definizione palette colori (primaria, secondaria, neutri)
- Definizione tipografia (scale, font primario/secondario)
- Creazione componenti UI base (button, card, form, header, footer)
- Prototipo interattivo in Figma per tutte le pagine
- Definizione stati interattivi (hover, focus, active)
- Design responsive (mobile-first)
- Export asset ottimizzati

**Tempo Stimato**: 16 ore  
**Costo**: €800

---

### 5. Implementazione Frontend

**Attività**:
- Setup progetto Astro con configurazione base
- Implementazione layout principale (header, footer)
- Implementazione homepage (hero, sezione aziende, CTA)
- Implementazione pagina "Il Gruppo" (chi siamo, valori, numeri)
- Implementazione pagina "Le Aziende" (grid card)
- Implementazione pagina "Contatti" (informazioni, mappa)
- Implementazione pagine legali (Privacy, Cookie Policy)
- Cookie banner globale
- Styling responsive completo
- Ottimizzazione performance (lazy load immagini, caching)

**Tempo Stimato**: 24 ore  
**Costo**: €1.200

---

### 6. Form Contatti e Integrazioni

**Attività**:
- Implementazione form contatti con validazione
- Setup invio email (SMTP o servizio esterno)
- Integrazione Google Maps (embed o API) o alternativa Open Street Maps
- Integrazione Google Analytics 4 o alternativa Umamy Analytics

**Tempo Stimato**: 6 ore  
**Costo**: €300

---

### 7. SEO e Ottimizzazioni

**Attività**:
- Configurazione meta tag per tutte le pagine
- Ottimizzazione H1-H6 e struttura semantica
- Generazione sitemap.xml
- Generazione robots.txt
- Generazione llms.txt
- Ottimizzazione URL pulite
- Test accessibilità base (WCAG 2.1 AA)
- Ottimizzazione Core Web Vitals

**Tempo Stimato**: 4 ore  
**Costo**: €200

---

### 8. Testing e QA

**Attività**:
- Test funzionalità su browser principali (Chrome, Firefox, Safari, Edge)
- Test responsive su dispositivi vari (mobile, tablet, desktop)
- Test form e validazione
- Test integrazioni (GA, Maps, email)
- Test performance (Lighthouse)
- Fix bug e ottimizzazioni

**Tempo Stimato**: 6 ore  
**Costo**: €300

---

### 9. Deployment e Configurazione Produzione

**Attività**:
- Build produzione ottimizzato
- Configurazione ambiente produzione su VPS
- Setup CI/CD base (se necessario)
- Configurazione backup automatici
- Test produzione completo
- Monitoraggio setup base

**Tempo Stimato**: 4 ore  
**Costo**: €200

---

### 10. Documentazione

**Attività**:
- Documentazione tecnica progetto
- Documentazione deployment
- README progetto

**Tempo Stimato**: 2 ore  
**Costo**: €100

## Note

- I costi non includono IVA
- I tempi sono stimati considerando che il cliente fornisca tutti gli asset e contenuti richiesti nei tempi previsti
- Eventuali modifiche sostanziali al design o alle funzionalità dopo l'approvazione dei wireframe potrebbero comportare costi aggiuntivi
- Il preventivo include un sito statico senza CMS: eventuali aggiornamenti futuri richiederanno intervento tecnico
- Costi infrastruttura hosting (VPS Hetzner) non inclusi nel preventivo (vedi documento **Infrastruttura e Hosting**)
